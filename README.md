# Raynott College - Demo React + Tailwind Project

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start dev server:
   ```
   npm run dev
   ```

This project uses Vite + React + TailwindCSS. It's a demo with client-side fake auth and placeholder images.

## Files
- src/components: Header, Footer
- src/pages: Landing, Auth, Course, Events, Faculty, About
- src/data/dummy.js: Dummy data for courses, events, faculty and alumni
